# Hombres Lobo de Castronegro 
Web app para jugar en persona a los hombres lobos de castronegro cuando te olvidas la baraja.

<img width="1796" height="601" alt="image" src="https://github.com/user-attachments/assets/7de4371c-e9d8-4c42-b9e6-3a4eee2423aa" />

Cada jugador se conecta con un móvil (o cualquier dispositivo) a la sala que crea el narrador, y así se pueden repartir de forma secreta los roles elegidos a cada jugador. También ayuda a llevar la partida, recordando el orden en el que despierta cada rol y que rol tiene cada jugador.

Funciona usando conexiones peer2peer con peerJS, así sale gratis hostearlo en github pages :)
